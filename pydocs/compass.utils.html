compass.utils.util.getListWithPossibility(lists)¶

Return list of item from list of list of identity item.

Parameters:lists – list of list of identity item.

Returns: list where for each first k elements, it should be the k most possible items.

Example: lists: [‘a’, ‘a’, ‘a’, ‘a’], [‘b’, ‘b’], [‘c’], the expected output is [‘a’, ‘b’, ‘c’, ‘a’, ‘a’, ‘b’, ‘a’]
